Student Attendance & Completion Analysis

Tools: SQL, Excel

Project Overview

This project analyses student attendance and completion patterns to identify engagement trends and potential risk indicators. SQL queries were used to analyse attendance patterns and student outcomes.

Dataset Structure

  • Student ID
  • Course
  • Attendance Percentage
  • Assessment Score
  • Completion Status

Example SQL Query


SELECT
course,
AVG(attendance) AS avg_attendance,
AVG(score) AS avg_score
FROM students
GROUP BY course;

Key Insights

  • Students with attendance above 85% show significantly higher completion rates.
  • Attendance below 70% is associated with increased risk of non-completion.
  • Early monitoring of attendance can help identify students needing support.
