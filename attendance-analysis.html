Student Attendance & Completion Analysis

Tools: SQL, Excel

1. Business Problem

Educational institutions often struggle to identify students who may be at risk of failing or dropping out. Attendance is frequently one of the earliest indicators of student engagement. This project explores whether attendance levels can help predict student completion outcomes and how reporting insights can help instructors intervene earlier.

2. Dataset Description

The dataset contains attendance and performance records collected across multiple teaching sessions. Each session includes multiple classes with approximately 15–20 students per class.

Estimated dataset size:

  • Sessions per year: 13
  • Classes per session: 2
  • Average students per class: 18
  • Students per year: ~468
  • Total dataset analysed: ~1400 student records (3 academic years)

Dataset Fields

  • student_id
  • session
  • course
  • attendance_percentage
  • assessment_score
  • completion_status

3. SQL Analysis

Average Attendance and Performance by Course

SELECT
course,
COUNT(student_id) AS total_students,
ROUND(AVG(attendance_percentage),2) AS avg_attendance,
ROUND(AVG(assessment_score),2) AS avg_score
FROM student_performance
GROUP BY course
ORDER BY avg_score DESC;

Identify At-Risk Students

SELECT
student_id,
session,
course,
attendance_percentage,
assessment_score
FROM student_performance
WHERE attendance_percentage < 70
ORDER BY attendance_percentage ASC;

Completion Rate by Attendance Band

SELECT
CASE
WHEN attendance_percentage >= 90 THEN '90–100%'
WHEN attendance_percentage >= 80 THEN '80–89%'
WHEN attendance_percentage >= 70 THEN '70–79%'
ELSE 'Below 70%'
END AS attendance_band,
COUNT(*) AS students,
AVG(CASE WHEN completion_status = 'Completed' THEN 1 ELSE 0 END)*100 AS completion_rate
FROM student_performance
GROUP BY attendance_band
ORDER BY attendance_band DESC;

4. Key Insights

  • Students with attendance above 85% show the highest completion rates.
  • Attendance below 70% strongly correlates with course non-completion.
  • Early attendance monitoring can help identify students requiring additional academic support.
  • Session-level reporting enables instructors and administrators to track engagement trends across multiple terms.

5. Reporting Dashboard

The analysis results were summarised in an Excel reporting dashboard designed to support operational monitoring of student engagement.

The dashboard includes:

  • Attendance distribution across sessions
  • Course completion rate overview
  • Average assessment score by attendance level
  • Identification of students at risk of non-completion
